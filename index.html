🏙️ What is EchoRoute?

  • Real-time urban noise monitoring for Mangalore, Karnataka
  • Citizens get proximity alerts for noise >70 dB within 1 km
  • Admins view live heatmaps & run noise-reduction simulations
  • Targets WHO guideline compliance for road-traffic noise
  • Hackathon 2026 — Dayanadha Sagar

🔧 Tech Stack

FastAPIUvicornPydantic React 18Leaflet GeoJSONOpenStreetMap Python 3.10+

📡 API Endpoints

  • GET /admin/thermal-data — GeoJSON heatmap dataset
  • GET /user/check-reroute?user_lat=&user_lon= — proximity alert
  • Algorithm: Haversine great-circle distance
  • Threshold: 70 dB (WHO guideline) • Radius: 1 km
  • Swagger UI at localhost:8000/docs

📊 Data Model

  • noise_level — float, simulated 40–90 dB
  • noise_status — Safe / Moderate / Harmful
  • traffic — low / medium / high
  • area — residential / pedestrian / normal
  • Format: GeoJSON FeatureCollection (WGS84)
